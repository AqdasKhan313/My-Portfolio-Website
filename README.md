# Aqdas Khan Portfolio Website

This is my personal portfolio website created to showcase skills, projects, education, and achievements. The website represents my journey in technology and development and works as an online profile where people can know more about me and my work.

## About the Website 

The portfolio includes information about me, my technical skills, projects, education, certifications, and contact details. I designed this website with a clean and responsive layout so that it works properly on different devices.

## Features

- Responsive design for mobiles,ipad etc.
- Modern and clean user interface
- About me section
- Skills section
- Projects showcase
- Education and achievements section
- Contact section with social links
- Smooth navigation and animations

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Projects Included

Some of the projects highlighted in this portfolio are:

- QR Generator with UPI Payment
- Digital Clock using Python
- Dashboard and Data Analytics Projects
- Web Development Projects
- Other learning and practice projects

## How to Run the Website

1. Download or clone this repository
2. Open the project folder
3. Run the `index.html` file in your browser

No additional setup or installation is required.
